# Frontend Components Documentation

This directory contains comprehensive documentation for all React components in the CoinDrop application. Components are organized by their functional domains.

## Savings Management
- [SavingsAccountManager](./SavingsAccountManager.md) - Core component for managing savings accounts

## Authentication
- UserLoginForm - User authentication component
- UserRegistrationForm - New user registration component
- ProtectedRoute - Route protection wrapper

## Budget Management
- BudgetManager - Budget tracking and management
- BudgetCard - Individual budget display
- BudgetCharts - Visual budget analytics
- CreateBudgetModal - Budget creation interface

## Dashboard
- DashboardManager - Main dashboard component
- DashboardBarChart - Budget visualization
- DashboardPieChart - Expense distribution
- DashboardQuickNavLinks - Navigation shortcuts
- DashboardUserGreetings - User welcome section
- DashboardUserShortAnalytics - Quick financial insights

## Education
- EducationManager - Financial education content
- EducationCard - Educational content display
- EducationFullDetailModal - Detailed content view
- EducationPostsGrid - Content organization
- UserEducationManager - User-specific content

## Profile
- ProfileManager - User profile management
- UserProfileDetails - Profile information display
- ProfileModal - Profile editing interface
- ImageUpload - Profile image management

## Common Components
- ErrorBoundary - Error handling wrapper
- LoadingSpinner - Loading state indicator

Each component documentation includes:
- Detailed description
- Props and state management
- Dependencies
- Usage examples
- Code implementation
- Styling information
- Performance considerations
